Grid system

Markup

How the grid system works. A row .row is mandatory, it can include as many columns .col-* as you want. Columns .col-* can be customized.

There are 12 relative widths per column and per screen sizes.

The row uccupies all the possible width, that depends on its container which is not included into this framework.

<div class="row">
    <div class="col-*"></div>
    ...
    <div class="col-*"></div>
</div>
